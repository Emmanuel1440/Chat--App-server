# 📱 Chat App - Frontend (Client-Side)

The user-facing client for the real-time chat application, styled to match WhatsApp's clean design. Built using React, Bootstrap, and WebSockets.

## 🚀 Features Implemented
* **Real-time Synchronization**: Instant message delivery and reaction updates using HTML5 WebSockets.
* **Smart Layout Alignment**: Your messages float on the right (WhatsApp Green), and incoming messages from other users align on the left (White).
* **Dynamic Bubble Shapes**: Styled asymmetrical borders mimicking actual WhatsApp bubble tails.
* **Media Sharing**: Preview and send image media uploads (< 10MB).
* **Emoji Support**: Reaction trays attached to each message bubble + a native emoji picker for typing.
* **Responsive Layout**: Fluid UI optimized for desktop and mobile layouts.
